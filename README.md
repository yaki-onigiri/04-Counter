## アプリ概要

シンプルなカウンターアプリです。
ボタンをクリックすることで数値を増減・リセットできます。

---

## アプリ画面

---

## How to Run

1. index.html をブラウザで開く
2. ボタンをクリックしてカウントを操作する

---

## Features（主な機能）

---

### 基本機能

・「+1」ボタンでカウントを増やす
・「-1」ボタンでカウントを減らす
・「リセット」ボタンでカウントを0に戻す
