# Dev-Log（詰まった箇所・解決方法）

---

## 26/04/13

---

### 目的

    カウンターアプリの基本機能（+1/-1/リセット）を実装する。

---

### 実装内容

    ・HTMLにて「現在の数値を表示する要素」を作成
    ・「+1」「-1」「リセット」ボタンを設置
    ・JSにてクリックイベントで数値を変更
    ・変更後の数値を画面に反映

---

### 実装の流れ

    1. HTMLに表示要素とボタンを追加
    2. JavaScriptでHTML要素を取得（getElementById）
    3. function変数 で数値を管理
    4. addEventListener でクリック処理を設定
    5. 値を変更し、textContent で画面更新

---

### 発生した問題①

同じ処理を何度も書いてしまう

---

#### 原因

    「表示更新処理」を毎回直接書いていた

---

#### 対処法

    関数にまとめて再利用するようにした

    function updateDisplay() {
        countEl.textContent = count;
    }

---

### 学んだこと

    ・「値の変更」と「画面更新」は別処理
    ・DOM操作はidの一致が前提
    ・同じ処理は関数化すると管理しやすい

    👉 基本の流れ：
    「イベント → 値変更 → 表示更新」

    👉 ミスしやすいポイント：
    「表示更新の書き忘れ」
